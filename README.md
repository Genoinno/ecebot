
# ECEBOT

A discord bot for the TOEC discord server. TOEC stands for: Tecnical One English Club, is an English Club that operates inside Vocational Highschool One Jakarta.

The bot uses mysql for the database and OpenHardwareMonitor for temperature monitoring.



## Features

- Common Management Commands
- Book Borrowing System
- Temperature Monitoring


## Environment Variables

To run this project, you will need to add the environment variables from `.example.env` to your `.env` file
